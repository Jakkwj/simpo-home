Chemostat​

请按照 AQUASIM 项目文件中的内容（而不是书中的内容），创建 BioModel, DataSet 和 Project。

提示

测量数据来自于 AQUASIM 的计算结果。

曝气计算方式：BioModel​

在原案例中，曝气计算的方式：

  • 在 BioModel 中添加了一行曝气反应：kLa⋅(SO2,sat−SO2)k_{L}a·(S_{O2,sat} - S_{O2})kL​a⋅(SO2,sat​−SO2​)，其中 kLak_{L}akL​a 为 100h−1100 h^{-1}100h−1
  • 在 Project.Activator 中的曝气池中激活这个反应，从而达到给系统持续供氧的目的

具体内容：BWTEE 2.3.4 Chemostat。

从计算结果中可见，Weighted NSE为0.99994，轻微的差异来自于：

  • S_O2（NSE 为 0.99922，0.28 vs 0.27995）
  • X_NO2（NSE 为 0.99998，0.05415 vs 0.05415）
  • X_DHO（NSE 为 0.99999，1.232 vs 1.23248）
  • 其他所有的 NSE 均为 1.0

这意味着 SIMPO 的模拟结果与 AQUASIM 中的结果几乎相同。

曝气计算方式：DataSet​

上述计算方式更适合简单的实验室模拟。 在实际工程中，曝气计算的方式通常需要在 DataSet 中进行。

SIMPO 中的曝气计算流程
SOTR​

  • 在标准条件下，即 20℃，1 个 atm 下空气的密度约为 1.204 kg/m³，质量占比约23.15%，即 1 m³ 空气中大约含有 1.204∗23.15=0.279kgO21.204 * 23.15 = 0.279 kgO_21.204∗23.15=0.279kgO2​。

  • 暂不考虑温度和气压变化的影响，则标准传氧速率（Standard Oxygen Transfer Rate，SOTR）：

    SOTR(mgO2/d)=0.279(kgO2/m3)∗Qair(m3/d)∗STOE(%)∗106 SOTR (mgO2/d) = 0.279 (kgO_2/m³) * Qair (m³/d) * STOE (\%) * 10^{6}SOTR(mgO2/d)=0.279(kgO2​/m3)∗Qair(m3/d)∗STOE(%)∗106

  • 其中：

    • QairQairQair为空气流量，m³/d
    • SOTE为标准传氧效率（Standard Oxygen Transfer Efficiency，SOTE），%
FOTR​

  • 实际反应器中通常是非标准温度的污水，所以要对 SOTRSOTRSOTR 进行修正得到实际传氧速率（Field Oxygen Transfer Rate，FOTR）：

    FOTR=dSO2/dt=SOTR∗θ(T−20)∗α∗(β∗SO2,sat−SO2)/SO2,sat,std/V FOTR = dS_{O2}/dt = SOTR * θ^{(T - 20)} * α * (β * S_{O2,sat} - S_{O2}) / S_{O2,sat,std} / VFOTR=dSO2​/dt=SOTR∗θ(T−20)∗α∗(β∗SO2,sat​−SO2​)/SO2,sat,std​/V

  • 其中：

    • θ 为温度影响因子
    • T 为温度，℃
    • SO2,sat,stdS_{O2,sat,std}SO2,sat,std​ 为标准条件下的清水中饱和溶解氧浓度，约 9.08~9.17 mg/L，工程上常取 9.08 mg/L
    • α 氧传递修正系数（Oxygen Transfer Ratio）：kLa(Wastewater)/kLa(CleanWater)k_{L}a(Wastewater)/k_{L}a(Clean Water)kL​a(Wastewater)/kL​a(CleanWater)
    • β 氧溶解修正系数（Oxygen Solubility Ratio）：SO2,sat(Wastewater)/SO2,sat(CleanWater)S_{O2,sat}(Wastewater)/S_{O2,sat}(Clean Water)SO2,sat​(Wastewater)/SO2,sat​(CleanWater)
    • V 为反应器体积，m³
FOTE​
  • 风机厂家会提供 SOTE-流量 标准条件曲线，同样需要修正为实际传氧效率（Field Oxygen Transfer Efficiency，FOTE）。在 SIMPO中，可以根据 SOTE-流量 标准条件曲线拟合下式，求解 a，b，c（这个公式保证了进气流量与 SOTE 是非线性衰减递增）。 FOTE=a∗(Qd)−b∗Hc=a∗(Qair/d/24)−b∗Hc FOTE = a * (Q_d)^{-b} * H^c = a * (Qair / d / 24)^{-b} * H^cFOTE=a∗(Qd​)−b∗Hc=a∗(Qair/d/24)−b∗Hc
  • 其中：
    • QdQ_dQd​ 为单头通气量，m³/h/个曝气头
    • d 为曝气头数量
    • H 为曝气头安装深度，m
求解方程​
  • 在暂不考虑温度和气压变化的影响时，合并上述3式可得: dSO2/dt=279∗Qair∗a∗(Qair/d/24)−b∗Hc∗α∗(β∗SO2,sat−SO2)/SO2,sat,std/V dS_{O2}/dt = 279 * Qair * a * (Qair / d / 24)^{-b} * H^c * α * (β * S_{O2,sat} - S_{O2}) / S_{O2,sat,std} / VdSO2​/dt=279∗Qair∗a∗(Qair/d/24)−b∗Hc∗α∗(β∗SO2,sat​−SO2​)/SO2,sat,std​/V
  • 注意此时，dSO2/dtdS_{O2}/dtdSO2​/dt 的单位为 mgO2/L/dmgO_2/L/dmgO2​/L/d。

在这个案例中，为了保持 kLak_{L}akL​a 为 100h−1100 h^{-1}100h−1，我们选取了以下这组参数（FOTE为14.1%）：

  • Qair：3215 m³/d
  • d（Tank 参数 Diffuser Number）：1
  • H（Tank 参数 Diffuser Depth）：0.5 m
  • α（Tank 参数 Alpha Factor）：0.55
  • β（Tank 参数 Beta Factor）：1
  • a（Pump 参数 SOTEa）：0.3259
  • b（Pump 参数 SOTEb）：0.1342
  • c（Pump 参数 SOTEc）：0.2567
  • V：0.1 m³
  • SO2,sat,stdS_{O2,sat,std}SO2,sat,std​：9.08 mg/L

则：

kLa=279∗3215∗0.3259∗(3215/1/24)−0.1342∗0.50.2567∗0.55/9.08/0.1=76812.63d−1 k_{L}a = 279 * 3215 * 0.3259 * (3215 / 1 / 24) ^ {-0.1342} * 0.5 ^ {0.2567} * 0.55 / 9.08 / 0.1 = 76812.63 d^{-1}kL​a=279∗3215∗0.3259∗(3215/1/24)−0.1342∗0.50.2567∗0.55/9.08/0.1=76812.63d−1

注意到案例中的氧气的单位为 mmol/L，则:

kLa=76812.63/32/24≈100h−1 k_{L}a = 76812.63 / 32 / 24 ≈ 100 h^{-1}kL​a=76812.63/32/24≈100h−1

具体内容：BWTEE 2.3.4 Chemostat。

从计算结果中可见，Weighted NSE为0.99232，轻微的差异来自于：

  • S_O2（NSE 为 0.90021，0.28 vs 0.27877）
  • X_NO2（NSE 为 0.99999，0.05415 vs 0.05416）
  • X_DHO（NSE 为 0.99999，1.232 vs 1.23246）
  • 其他所有的 NSE 均为 1.0

这意味着，这种方式的模拟结果也与在 AQUASIM 中的结果几乎相同。

总结​

在实际使用中，可以根据实际情况选择不同的方式进行曝气计算：

1. 在实验室中进行简单模拟：

  • DataSet.Tank 使用默认参数（即 d 为 0，H 为 0，α 为 1，β 为 1）即可；
  • 在 BioModel 中添加了一行曝气反应：kLa⋅(SO2,sat−SO2)k_{L}a·(S_{O2,sat} - S_{O2})kL​a⋅(SO2,sat​−SO2​)；
  • 在 Project.Activator 中的曝气池中激活这个反应。

2. 在工程项目中进行更精确的模拟：

  • 根据厂家提供的 SOTE-流量 标准条件曲线，在 SIMPO 直接拟合求解参数 a，b，c；
  • 将 DataSet.Pump 连接到 DataSet.Tank，设置进气风量 Qair；
  • 根据实际情况设置 DataSet.Tank 的参数：d，H，α，β。
