SIMPO

水和污水处理过程数值模拟专业科学计算SaaS平台

SIMPO 结合了用户友好性和强大的灵活性，是一个用于水和污水处理过程模拟的开放的现代云平台。 SIMPO 旨在降低建模的学习和研究门槛，提高开发效率，提供科学的算法和评估方法，最终实现可持续发展的智能大数据平台，为所有人提供共创合作、分享和开放教育资源（OER）的平台。

易用

让您专注模型推导和实验数据，我们将处理计算过程：

  • Through the user-friendly front-end interface, model building can be done quickly without any programming.
  • The input of models and data can be accomplished through: 1) importing xlsx files, 2) copying public models/data, or 3) creating new ones directly.
  • SIMPO will automatically generate a process flow diagram and a series of graphs of the calculation results, users can export the result data for further processing.

强大且灵活

Provides various calculation engines and functions:

  • Calculation engines includes: simulation, sensitivity analysis, uncertainty analysis and parameter estimation.
  • Models and data can be combined in any way users see fit, so it's very easy to identify the optimal model for a given dataset and evaluate a model's robustness and generalizability.
  • Models and data can be released in multiple versions, making it easy to determine the optimal model structure and parameter.

开放

An open-minded platform that provides OER for everyone:

  • Everyone is encouraged to contribute code to co-create algorithms.
  • Works on Windows, Linux, and Mac, it can leverage the power of supercomputers, as their systems are usually Linux-based.
  • 模型， 数据和计算结果能发布成公开版本，从而将研究成果分享给整个研究社区。
  • SIMPO provides OER to all researchers and students. This endeavor aligns with the United Nations' Sustainable Development Goal 4: Quality Education.

可追随

避免作弊：

  • All results are calculated according to open-source code, there is no possibility of modifying the calculation results.
  • All model submissions can be calculated and published publicly, thereby precluding the possibility of algorithmic cheating.

欢迎共创

  • 加入我们使用强大的AI工具来构建和增强 Paper Resource。您的贡献可以显著提升和扩展资源库，使其对整个社区更有价值。
  • 感谢您的支持与合作，欢迎在以下地址进行 fork 和 pull：simpo-home
