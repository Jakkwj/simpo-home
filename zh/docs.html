快速开始

SIMPO的3个核心元素是：BioModel （模型）, DataSet （数据）和 Project （解法）。

完整计算过程的步骤如下：

  • 步骤1: 在你的计算机上安装计算客户端，然后注册并登录 Dashborad。
  • 步骤2: 创建一个 BioModel。
  • 步骤3: 创建一个 DataSet。
  • 步骤4: 创建一个 Project，将 BioModel 和 DataSet 组合在一起，然后选择一个计算引擎来求解。
克隆即可开始

通过克隆公开 Project，我们就可以直接开始计算，而无需经过上述这些步骤。但请记住，在此之前先安装计算客户端并登录。

最简单的方式​

共享研究成果是 SIMPO 的主要功能之一，因此从公开资源库中克隆一个项目是最简单的方法。该操作将克隆相应的 BioMode、Dataset 和计算结果。

  • 克隆一个公开项目。

  • 检查计算结果（您可以更改某些参数并重新计算）。

尝试ASM3​

SIMPO 有很多公开的经典模型，我们可以克隆并使用它们。这里我们以ASM3为一个简单的例子。

  • 从公开资源库克隆 ASM3 到个人 BioModel 资源库。

  • 创建一个新的项目，包含一个 ASM3 生物模型和一个空的数据集，然后你会得到（这些设置是默认创建的，但是当然您可以更改它们）：

    • 一个 CSTR 反应器，其 Targets 与 ASM3 的 Components 完全相同。
    • 所有 Targets 的初始值均设定为 200。
    • 计算时长为 2 天，间隔为 0.2 天。
    • ASM3 的所有 Parameters 的 Evaluation 列（在 Project Detail Variable 选项卡中）均设为 “True”。

  • 然后您就可以开始计算了。
