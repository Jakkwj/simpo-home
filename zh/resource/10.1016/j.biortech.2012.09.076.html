Simulation of the performance of aerobic granular sludge SBR using modified ASM3 model

DOI: 10.1016/j.biortech.2012.09.076

作者: Man Zhou, Jianyu Gong, Changzhu Yang, Wenhong Pu

活性污泥模型3号 (ASM3) 经过修正，用于描述好氧颗粒污泥 SBR 中的生物反应。修正后的模型考虑了同步储存与生长、硝化和反硝化过程。分析了出水 COD、NH4+-N 和 TN 对化学计量和动力学系数的敏感性。模型选用了一组结合文献数据获得的标准参数。利用典型周期内 COD、NH4+-N 和 TN 的时间变化实验结果验证了 ASM3 模型。验证结果表明，建立的模型适用于模拟好氧颗粒污泥 SBR 的性能。此外，还比较了修正后的 ASM3 和原始 ASM3 对基质去除的实测值与预测值。验证和比较结果表明，修正后的 ASM3 模型能更好、更具机理地描述好氧颗粒污泥 SBR。

Q：这篇论文试图解决什么问题？

Claude Sonnet 4.6:

本文试图解决原始ASM3模型在准确模拟好氧颗粒污泥序批式反应器（SBR）中生物转化过程方面的局限性。具体而言，原始ASM3存在两个主要缺陷：

  • 饱食阶段仅储存的假设：原始ASM3假设所有易生物降解底物（SSS_SSS​）首先以内部细胞聚合物（XSTOX_{STO}XSTO​）的形式储存，然后再用于生长，忽略了底物直接生长的同步过程。这一假设无法捕捉好氧颗粒污泥SBR中的饱食-饥饿行为，导致对生物量生长速率和内部储存聚合物水平的预测不准确。

  • 单步硝化和反硝化：原始ASM3将硝化和反硝化视为单步过程，忽略了亚硝酸盐（NO2−NO_2^-NO2−​）作为中间产物的作用。在好氧颗粒污泥系统中，由于氧气限制、高温、污泥停留时间不足和游离氨抑制等因素，亚硝酸盐可能显著积累，使单步假设不够充分。

这些局限性导致原始ASM3对好氧颗粒污泥SBR出水COD、NH4+NH_4^+NH4+​–N和TN的模拟精度较差，其拟合优度分别仅为56.5%、52.6%和58.9%。

Gemini 3.1 Pro:

  • 原始的ASM3模型在描述序批式反应器（SBR）中好氧颗粒污泥的性能时存在局限性。
  • ASM3假设所有易生物降解的底物首先被吸收并储存为内部细胞聚合物，然后再用于生长，这无法准确描述SBR中异养菌的“饱食-饥饿”行为（即同步储能与生长）。
  • ASM3假设硝化和反硝化都是单步过程，忽略了亚硝酸盐（NO2−NO_2^-NO2−​）作为中间产物的作用，而这在亚硝酸盐容易积累的好氧颗粒污泥系统中是非常重要的。

Gemini 3.0 Pro:

本文试图解决标准活性污泥模型3号 (ASM3) 在应用于序批式反应器 (SBR) 中的好氧颗粒污泥 (AGS) 系统时的局限性。具体包括：

  • 顺序与同步过程：ASM3 假设易生物降解基质首先被储存为内部聚合物，然后再用于生长（顺序过程）。然而，在 AGS 系统中，特别是在饱食 (feast) 条件下，储存和生长通常是同步发生的。
  • 亚硝酸盐积累：ASM3 将硝化和反硝化模拟为单步过程，忽略了作为中间产物的亚硝酸盐 (NO2−NO_2^-NO2−​)。由于氧扩散限制，AGS 系统中经常出现亚硝酸盐积累，这是原始模型无法描述的。

Q：论文如何解决这个问题？

Claude Sonnet 4.6:

本文通过开发一个改进的ASM3模型来解决上述问题，该模型在MATLAB中实现，包含两项关键扩展：

改进1：同步储存与生长​

  • 与原始ASM3的顺序储存-再生长假设不同，改进模型引入了异养菌（XHX_HXH​）对底物的好氧/缺氧同步储存与生长过程。
  • 在饱食条件下，SSS_SSS​同时用于储存（生成XSTOX_{STO}XSTO​）和XHX_HXH​的直接生长。
  • 在饥饿条件下（SSS_SSS​耗尽时），XSTOX_{STO}XSTO​作为碳源用于二次生长。
  • 新增过程包括：好氧/缺氧条件下对SSS_SSS​的储存、好氧/缺氧条件下对SSS_SSS​的生长、好氧/缺氧条件下对XSTOX_{STO}XSTO​的生长，以及相应的呼吸过程。

改进2：两步硝化与反硝化​

  • 亚硝酸盐（SNO2S_{NO_2}SNO2​​）作为新的模型组分和中间产物被引入。
  • 原始单一自养生物量（XAX_AXA​）被替换为两个独立群体：氨氧化菌（AOB，XAOBX_{AOB}XAOB​）和亚硝酸盐氧化菌（NOB，XNOBX_{NOB}XNOB​）。
  • 硝化分为两步：NH4+→NO2−NH_4^+ \rightarrow NO_2^-NH4+​→NO2−​（由AOB完成）和NO2−→NO3−NO_2^- \rightarrow NO_3^-NO2−​→NO3−​（由NOB完成）。
  • 反硝化也分为两步：异养菌将NO3−NO_3^-NO3−​还原为NO2−NO_2^-NO2−​，再将NO2−NO_2^-NO2−​还原为N2N_2N2​。

模型结构​

  • 改进模型包含13个组分和22个微生物过程，以化学计量矩阵格式呈现。
  • 模型在MATLAB中实现（未使用标准模拟软件）。

灵敏度分析与校准​

  • 通过将所有动力学和化学计量参数各变化±10%，进行灵敏度分析，识别对出水COD、NH4+NH_4^+NH4+​–N和TN影响最大的参数。
  • 对11个关键参数（YH,O2Y_{H,O2}YH,O2​、KSK_SKS​、μH,STO\mu_{H,STO}μH,STO​、μH\mu_HμH​、bSTO,O2b_{STO,O2}bSTO,O2​、μAOB\mu_{AOB}μAOB​、KAOB,O2K_{AOB,O2}KAOB,O2​、ηNO3\eta_{NO3}ηNO3​、ηNO2\eta_{NO2}ηNO2​、bNOBb_{NOB}bNOB​、KNOB,O2K_{NOB,O2}KNOB,O2​）进行迭代校准。

Gemini 3.1 Pro:

  • 对ASM3模型进行了修改，引入了异养细菌对有机底物的同步好氧/缺氧储能与生长的概念。
  • 引入了两步硝化和反硝化过程，将亚硝酸盐作为中间化合物包含在内。

Gemini 3.0 Pro:

作者通过以下扩展开发了修正的 ASM3 模型：

  • 同步储存和生长：修改模型以允许异养细菌对有机基质 (SSS_SSS​) 进行同步好氧/缺氧储存和生长。

Q：论文做了哪些实验？

Claude Sonnet 4.6:

实验在实验室规模的好氧颗粒污泥SBR中进行，具体设置和步骤如下：

反应器设置​

  • 有机玻璃SBR，有效容积8 L，高度1.6 m，内径0.08 m。
  • 每周期体积交换比：50%。
  • 每周期：进水2 min → 曝气354 min → 沉淀3 min → 出水1 min。
  • 通过底部微孔曝气器供气；通过气流调节器控制DO。
  • 温度维持在20 ± 1°C。

接种污泥与颗粒培养​

  • 好氧颗粒由取自武汉市污水处理厂的活性污泥培养而成。
  • 稳态时MLSS维持在约5.0 g/L。
  • 稳态时颗粒近似球形，直径0.6–4.0 mm。

合成废水组成​

  • 碳源：NaAc·3H₂O（1279–1860 mg/L）
  • 氮源：NH₄Cl（175–255 mg/L）
  • 其他营养盐：KH2PO4KH_2PO_4KH2​PO4​、MgSO4MgSO_4MgSO4​、Fe2(SO4)3Fe_2(SO_4)_3Fe2​(SO4​)3​、CaCl2⋅H2OCaCl_2·H_2OCaCl2​⋅H2​O

分析测定​

  • COD、NH4+NH_4^+NH4+​–N、TN和MLSS按标准方法（APHA, 1995）测定。
  • DO用溶解氧仪（LEICI JPB-607）测定。
  • pH用微处理器pH计（pHS-3B）测定。
  • SSS_SSS​、XSX_SXS​和SIS_ISI​按Mamais et al. (1993)和Melcer et al. (2004)的方法测定。
  • 所有测定均在稳态条件下进行。

模型验证数据采集​

  • 采集**典型SBR周期（6小时）**内COD、NH4+NH_4^+NH4+​–N和TN浓度的时间序列数据，用于模型验证（图3）。
  • 采集**长期（30天）**出水COD和NH4+NH_4^+NH4+​–N数据，在变化进水条件下验证模型预测性能（图4）。

Q：有什么可以进一步探索的点？

Claude Sonnet 4.6:

  • 提高TN模拟精度：单周期验证中TN的拟合优度仅为92.4%，低于COD（98.9%）和NH4+NH_4^+NH4+​–N（97.9%）。进一步完善反硝化动力学，并将氧化亚氮（N2ON_2ON2​O）作为中间产物纳入模型，可能有助于提高TN预测精度。

  • 纳入磷去除过程：当前改进的ASM3模型未考虑生物除磷（EBPR）。好氧颗粒污泥系统已被证明可同步去除COD、氮和磷（如de Kreuk et al., 2005a所示）。将磷积累菌（PAOs）纳入模型将使其更加全面。

  • 颗粒结构与扩散效应建模：当前模型将颗粒视为均质系统，未明确考虑颗粒内部氧气和底物的扩散梯度。引入生物膜/颗粒扩散模型（如多尺度个体模型）可提高精度，尤其在氧气限制条件下。

  • 温度效应：模型仅在20°C下进行校准和验证。引入温度修正因子（Arrhenius型）将允许在不同温度条件下进行模拟，这对实际应用非常重要。

  • 抑制效应：模型未明确包含游离氨（FA）和游离亚硝酸（FNA）对AOB和NOB的抑制，而这些是好氧颗粒污泥系统中亚硝酸盐积累的重要影响因素。

  • 实际市政/工业废水的应用验证：模型仅用合成废水进行了验证。在实际应用前，需要用成分复杂的真实市政或工业废水进行验证。

  • 长期TN预测：本文仅展示了COD和NH4+NH_4^+NH4+​–N的长期（30天）预测（图4），未包含TN的长期预测与验证，这一方面有待进一步探索。

  • 模型参数不确定性分析：本文仅进行了确定性灵敏度分析。开展正式的不确定性/可识别性分析（如蒙特卡洛模拟、MCMC方法）可提供更稳健的参数估计和置信区间。

  • 颗粒形成动力学扩展：当前模型假设颗粒形态处于稳态。对好氧颗粒动态形成和演变过程进行建模，对于理解启动阶段和运行优化具有重要价值。

Gemini 3.1 Pro:

  • 模拟的TN浓度与测量值之间存在一定偏差，表明颗粒内部的反硝化过程或同步硝化反硝化（SND）机制可以进一步细化。
  • 该模型可以进一步扩展以包含除磷过程（例如，整合ASM2d的特征）。
  • 可以将微生物的空间分布和好氧颗粒3D结构内的传质限制（扩散）纳入其中，以建立更具机制性的多尺度模型。

  • 温室气体排放：论文指出一氧化氮 (NONONO) 和氧化亚氮 (N2ON_2ON2​O) 在建模中常被忽略。虽然文中提到了反应步骤，但当前模型并未明确跟踪 NONONO 和 N2ON_2ON2​O 作为动态状态变量以预测排放。
  • 实际废水应用：本研究使用的是合成废水进行验证。测试模型在具有不同成分的实际市政或工业废水中的稳健性将非常有价值。
  • 温度影响：模型参数设定为 20°C。探索不同温度条件下模型的性能和参数敏感性对于全规模应用具有相关性。

Q： 总结一下论文的主要内容。

Claude Sonnet 4.6:

本文提出了一个改进的ASM3模型，用于模拟好氧颗粒污泥序批式反应器（SBR）中的生物转化过程。主要内容如下：

1. 研究背景与动机​

  • 好氧颗粒污泥SBR是一种具有结构紧凑、生物量保留率高、沉降性好等优点的新型废水处理技术。
  • 原始ASM3模型对好氧颗粒污泥系统存在两个关键局限：（i）假设顺序储存-再生长（而非同步），（ii）将硝化/反硝化视为单步过程，忽略亚硝酸盐积累。

2. 模型开发​

  • 改进模型包含13个组分和22个微生物过程。
  • 改进1：在饱食-饥饿条件下，异养菌对SSS_SSS​和XSTOX_{STO}XSTO​进行好氧/缺氧同步储存与生长。
  • 改进2：两步硝化（AOB：NH4+→NO2−NH_4^+ \rightarrow NO_2^-NH4+​→NO2−​；NOB：NO2−→NO3−NO_2^- \rightarrow NO_3^-NO2−​→NO3−​）和两步反硝化（NO3−→NO2−→N2NO_3^- \rightarrow NO_2^- \rightarrow N_2NO3−​→NO2−​→N2​），SNO2S_{NO_2}SNO2​​作为新的模型组分。
  • 模型在MATLAB中实现，以化学计量矩阵格式呈现。

3. 灵敏度分析​

  • 对所有动力学和化学计量参数进行±10%变化的灵敏度分析。
  • COD的关键敏感参数：YH,O2Y_{H,O2}YH,O2​、YH,STOY_{H,STO}YH,STO​、KXK_XKX​、KSK_SKS​、μH,STO\mu_{H,STO}μH,STO​。
  • NH4+NH_4^+NH4+​–N的关键敏感参数：YH,NO3Y_{H,NO3}YH,NO3​、YAOBY_{AOB}YAOB​、KO2K_{O2}KO2​、μAOB\mu_{AOB}μAOB​、KAOB,NH4K_{AOB,NH4}KAOB,NH4​、KAOB,O2K_{AOB,O2}KAOB,O2​。
  • TN的关键敏感参数：YH,NO3Y_{H,NO3}YH,NO3​、YAOBY_{AOB}YAOB​、ηNO3\eta_{NO3}ηNO3​、ηNO2\eta_{NO2}ηNO2​、KNH4K_{NH4}KNH4​、bSTO,NO3b_{STO,NO3}bSTO,NO3​、ηNend\eta_{Nend}ηNend​。

4. 模型校准​

  • 基于灵敏度分析结果，对11个关键参数进行迭代校准：YH,O2Y_{H,O2}YH,O2​、KSK_SKS​、μH,STO\mu_{H,STO}μH,STO​、μH\mu_HμH​、bSTO,O2b_{STO,O2}bSTO,O2​、μAOB\mu_{AOB}μAOB​、KAOB,O2K_{AOB,O2}KAOB,O2​、ηNO3\eta_{NO3}ηNO3​、ηNO2\eta_{NO2}ηNO2​、bNOBb_{NOB}bNOB​、KNOB,O2K_{NOB,O2}KNOB,O2​。

5. 模型验证​

  • 以典型SBR周期内COD、NH4+NH_4^+NH4+​–N和TN的时间序列数据进行模型验证。
  • 拟合优度：COD = 98.9%，NH4+NH_4^+NH4+​–N = 97.9%，TN = 92.4%。
  • 长期（30天）模拟显示COD拟合优度为99.7%，NH4+NH_4^+NH4+​–N偏差在**0.8%**以内。

6. 与原始ASM3的比较​

  • 原始ASM3的拟合优度仅为COD 56.5%、NH4+NH_4^+NH4+​–N 52.6%、TN 58.9%。
  • 改进ASM3显著优于原始模型，证明同步储存-生长和两步硝化-反硝化改进对于准确模拟好氧颗粒污泥SBR至关重要。
