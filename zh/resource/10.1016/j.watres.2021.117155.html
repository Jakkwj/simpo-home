Development of a kinetic model to evaluate thiosulfate-driven denitrification and anammox (TDDA) process

DOI: 10.1016/j.watres.2021.117155

作者: Yang-Fan Deng, Wen-Tao Tang, Hao Huang, Jin Qian, Di Wu, Guang-Hao Chen

最近，硫驱动反硝化与厌氧氨氧化（Anammox）工艺的耦合作为一种有前景的替代脱氮技术已被广泛研究。大多数研究调查了工艺的可行性并监测了群落动态。然而，基于数学模型对这一新型硫-氮循环生物工艺的深入理解以及对不同微生物之间复杂相互作用的阐明尚未实现。为了填补这一空白，我们开发了一个动力学模型（包含7个生物过程，12个变量和19个参数）来评估单一反应器中的硫（硫代硫酸盐）驱动反硝化和厌氧氨氧化（TDDA）工艺。该过程中使用的参数是通过拟合实验数据分别估算的。随后，模型在不同条件下进行了进一步验证，结果表明开发的模型能够描述TDDA系统中氮和硫转化的动态行为。新开发的支路硫代硫酸盐氧化模型也通过宏基因组分析得到了验证。利用开发的模型，我们 i) 考察了在不同底物条件下稳态时硫氧化细菌与厌氧氨氧化细菌之间的相互作用，以证明TDDA的可靠性，以及 ii) 从实际实施的角度评估了TDDA工艺的可行性和运行。我们的结果将有助于进一步探索这一新型S-N循环生物工艺的重要意义，并指导其未来的应用。

Q：这篇论文试图解决什么问题？

Claude Sonnet 4.6:

  • 缺乏针对TDDA集成系统的数学模型：尽管硫驱动反硝化与厌氧氨氧化（anammox）的集成工艺已被广泛研究，但目前尚无针对单反应器中硫代硫酸盐驱动反硝化与厌氧氨氧化（TDDA）联合工艺的动力学数学模型。
  • 硫代硫酸盐氧化途径认识不足：以往简化模型假设硫代硫酸盐直接被氧化为硫酸盐（Sox途径），不产生中间产物，但在TDDA系统中观察到显著的元素硫（S0S^0S0）积累，表明存在不同的（分支）氧化途径，尚未被建模描述。
  • SOB与厌氧氨氧化菌之间的相互作用不明：硫氧化细菌（SOB）与厌氧氨氧化菌之间的复杂相互作用——包括对NO2−NO_2^-NO2−​作为电子受体的竞争，以及部分反硝化向厌氧氨氧化供给NO2−NO_2^-NO2−​——尚未得到定量理解。
  • 缺乏工艺评估与优化的实用工具：目前没有基于模型的工具来评估TDDA工艺的可行性、评价关键操作参数（如S/N比、NH4+/NO3−NH_4^+/NO_3^-NH4+​/NO3−​比）的影响，以及指导处理含高浓度硝酸盐和铵盐工业废水时的硫代硫酸盐投加量。

Gemini 3.1 Pro:

  • 缺乏基于数学模型对硫-氮循环生物工艺（特别是硫驱动反硝化与厌氧氨氧化结合工艺）的深入理解。
  • 未能阐明TDDA工艺中不同微生物（硫氧化细菌(SOB)和厌氧氨氧化细菌）之间复杂的相互作用。
  • 缺乏针对整合的硫（硫代硫酸盐）驱动反硝化与厌氧氨氧化（TDDA）系统的综合动力学模型。

Gemini 3.0 Pro:

本文试图解决对硫驱动反硝化与厌氧氨氧化（TDDA）耦合工艺缺乏深入理解的问题。具体而言：

  • 以往的研究主要集中在可行性和群落动态上，缺乏数学模型研究。
  • 目前没有现成的模型能阐明单一反应器中硫氧化菌（SOB）和厌氧氨氧化菌（AnAOB）之间复杂的相互作用。
  • 现有的硫驱动反硝化模型通常较为简化，未考虑单质硫（S0S^0S0）作为中间产物的形成或与厌氧氨氧化的耦合。
  • 文章旨在提供一种工具来评估工艺稳定性、优化运行，并理解两种细菌群落之间对底物（特别是亚硝酸盐）的竞争。

Q：论文如何解决这个问题？

Claude Sonnet 4.6:

1. 开发新型动力学模型（TDDA模型）​

  • 建立了包含7个生化过程、12个变量和19个参数的TDDA系统动力学模型。
  • 模型涉及两类微生物：硫氧化细菌（XSOBX_{SOB}XSOB​）和厌氧氨氧化菌（XAMXX_{AMX}XAMX​），采用双项Monod型动力学表达底物摄取速率。
  • 提出了分支硫代硫酸盐氧化模型：
    • 第一步：S2O32−S_2O_3^{2-}S2​O32−​被氧化为等比例的SO42−SO_4^{2-}SO42−​和S0S^0S0（每分子释放2个电子）：S2O32−+H2O→SO42−+S0+2H++2e−S_2O_3^{2-} + H_2O \rightarrow SO_4^{2-} + S^0 + 2H^+ + 2e^-S2​O32−​+H2​O→SO42−​+S0+2H++2e−
    • 第二步：S0S^0S0进一步被氧化为SO42−SO_4^{2-}SO42−​（每分子释放6个电子）：S0+4H2O→SO42−+8H++6e−S^0 + 4H_2O \rightarrow SO_4^{2-} + 8H^+ + 6e^-S0+4H2​O→SO42−​+8H++6e−
  • 采用两步反硝化模型：NO3−→NO2−→N2NO_3^- \rightarrow NO_2^- \rightarrow N_2NO3−​→NO2−​→N2​。
  • 7个生化过程包括：(1) 硝酸盐还原耦合硫代硫酸盐氧化，(2) 亚硝酸盐还原耦合硫代硫酸盐氧化，(3) 硝酸盐还原耦合元素硫氧化，(4) 亚硝酸盐还原耦合元素硫氧化，(5) 厌氧氨氧化，(6) XSOBX_{SOB}XSOB​衰减，(7) XAMXX_{AMX}XAMX​衰减。

2. 模型校准与参数估计​

  • 使用AQUASIM和MATLAB R2018a，通过拟合批次实验数据进行参数估计，最小化最小二乘代价函数：J(θ)=min⁡∑t=1N(y(t)−y(t,θ))2J(\theta) = \min \sum_{t=1}^{N}(y(t) - y(t,\theta))^2J(θ)=min∑t=1N​(y(t)−y(t,θ))2
  • 利用Fisher信息矩阵（FIM）进行可识别性分析，计算95%置信区间。
  • 估计了9个参数（μSOB1\mu^1_{SOB}μSOB1​、μSOB2\mu^2_{SOB}μSOB2​、μSOB3\mu^3_{SOB}μSOB3​、μSOB4\mu^4_{SOB}μSOB4​、KS2O3K_{S_2O_3}KS2​O3​​、KS0K_{S^0}KS0​、YSOB1Y_{SOB1}YSOB1​、YSOB2Y_{SOB2}YSOB2​、μAMX\mu_{AMX}μAMX​），其余10个参数来自文献。

3. 不同条件下的模型验证​

  • 在两组独立批次实验（B1和B2）下验证模型，分别获得R2>0.84R^2 > 0.84R2>0.84和R2>0.86R^2 > 0.86R2>0.86，证明模型能够描述氮和硫的动态转化。

4. 宏基因组分析验证硫代硫酸盐氧化途径​

  • 对UASB反应器污泥进行宏基因组测序（Illumina MiSeq 2500），验证分支氧化途径。
  • 定量分析关键基因丰度（SoxXA、SoxYZ、SoxB、SoxCD、DsrAB、AprAB、Sat），确认分支氧化途径（不完整Sox + DsrAB/AprAB/Sat）的主导地位。

5. 稳态模拟评估SOB与厌氧氨氧化菌的相互作用​

  • 在不同S/N（0.5–3.5 gS/gN）和NH4+/NO3−NH_4^+/NO_3^-NH4+​/NO3−​（0.3–1.2 gN/gN）比值下模拟TDDA系统稳态，评估生物量分布、脱氮途径及工艺可行性。

Gemini 3.0 Pro:

  • 模型开发：构建了一个包含7个生物过程、12个变量和19个参数的动力学模型。它明确模拟了硫代硫酸盐的两步氧化过程：硫代硫酸盐（S2O32−S_2O_3^{2-}S2​O32−​）转化为单质硫（S0S^0S0）和硫酸盐（SO42−SO_4^{2-}SO42−​），然后是S0S^0S0转化为SO42−SO_4^{2-}SO42−​。

1. UASB反应器长期运行实验​

  • 在有效容积为2.0 L的上流式厌氧污泥床（UASB）反应器中运行205天（第221–425天），分3个阶段逐步提高底物浓度：
    • 第1阶段（第221–268天）：进水NO3−NO_3^-NO3−​-N ~101.8 mgN/L，NH4+NH_4^+NH4+​-N ~97.2 mgN/L，S2O32−S_2O_3^{2-}S2​O32−​-S ~201.4 mgS/L
    • 第2阶段（第269–340天）：进水NO3−NO_3^-NO3−​-N ~123.5 mgN/L，NH4+NH_4^+NH4+​-N ~118.9 mgN/L，S2O32−S_2O_3^{2-}S2​O32−​-S ~238.7 mgS/L
    • 第3阶段（第341–425天）：进水NO3−NO_3^-NO3−​-N ~141.9 mgN/L，NH4+NH_4^+NH4+​-N ~140.3 mgN/L，S2O32−S_2O_3^{2-}S2​O32−​-S ~276.1 mgS/L
  • 温度：30 ± 1°C；HRT：全程12 h。
  • TN去除率达75.68%–82.01%；硫代硫酸盐去除率100%。

Q：有什么可以进一步探索的点？

Claude Sonnet 4.6:

  • 以基因组为中心的宏基因组学研究：文章指出，对宏基因组组装的scaffolds进行分箱以重建单个物种基因组的工作有待进一步探索，这种以基因组为中心的方法将有助于更好地理解TDDA系统的系统发育多样性和代谢途径。

  • 批次与连续反应器生物量产率系数差异的机制：YSOB1Y_{SOB1}YSOB1​和YSOB2Y_{SOB2}YSOB2​在批次实验和连续反应器中的差异原因尚不明确，可能与连续反应器中硫代硫酸盐被其他电子受体（如氧气）化学或生物氧化有关，值得深入研究。

  • 亚硝酸盐抑制效应：模型未考虑NO2−NO_2^-NO2−​对反硝化和厌氧氨氧化的抑制效应（实验中最高亚硝酸盐浓度为40.0 mgN/L，pH 8.0）。未来研究可在亚硝酸盐浓度更高的系统中引入亚硝酸盐抑制动力学。

  • 反硝化中间产物NO和N2ON_2ON2​O：模型将反硝化简化为两步过程（NO3−→NO2−→N2NO_3^- \rightarrow NO_2^- \rightarrow N_2NO3−​→NO2−​→N2​），未考虑NO和N2ON_2ON2​O中间产物。纳入这些中间产物可提高模型精度，对温室气体排放评估也具有重要意义。

  • 与异养反硝化的耦合：文章简要讨论了异养反硝化与厌氧氨氧化耦合的挑战（异养菌过度生长风险）。可开发包含异养反硝化的综合模型，用于处理同时含有机物和无机氮的废水。

  • 中试和全规模应用验证：模型在实验室规模下开发和验证，其在不同环境条件（温度、pH、进水水质波动）下的中试和全规模TDDA系统适用性有待评估。

  • 扩展至其他硫电子供体：当前模型聚焦于硫代硫酸盐（S2O32−S_2O_3^{2-}S2​O32−​）和元素硫（S0S^0S0），将模型扩展至其他硫化合物（如S2−S^{2-}S2−、SO32−SO_3^{2-}SO32−​、S4O62−S_4O_6^{2-}S4​O62−​）将拓宽其适用范围。

  • 硫代硫酸盐投加量的动态优化控制：虽然模型提供了评估S/N和NH4+/NO3−NH_4^+/NO_3^-NH4+​/NO3−​比值的框架，但针对进水水质波动的实时模型预测控制策略仍有待开发。

  • 生物膜和颗粒污泥建模：当前模型基于悬浮生长动力学，将其扩展至生物膜或颗粒污泥构型（如颗粒污泥UASB）需要引入传质限制和空间梯度。

Gemini 3.1 Pro:

  • 对宏基因组组装获得的支架（scaffolds）进行分箱（binning），以重建单个物种的基因组，从而更好地了解TDDA系统的系统发育多样性和代谢途径。
  • 调查批次反应器和连续反应器之间生物量产率系数存在差异的原因，例如连续反应器中可能存在其他电子受体（如氧气）对硫代硫酸盐的化学或生物氧化。
  • TDDA工艺后的进一步深度处理，因为根据排放标准和进水比例，系统中的NH4+NH_4^+NH4+​可能会过剩。

Gemini 3.0 Pro:

  • 长期连续模拟：文章指出，长期验证的结果不如动态批次模拟令人满意，这可能是由于生物量产率系数的变化或连续反应器中硫代硫酸盐的化学氧化。这种差异需要进一步调查。
  • 异养反硝化：模型忽略了异养反硝化（假设没有有机物）。在处理实际废水（如工业废水）时，有机物可能存在，需要扩展模型。

Q： 总结一下论文的主要内容。

Claude Sonnet 4.6:

本文开发、校准、验证并应用了硫代硫酸盐驱动反硝化与厌氧氨氧化（TDDA）工艺的动力学数学模型，主要贡献和发现如下：

1. 新型动力学模型的开发​

  • 建立了包含7个生化过程、12个变量和19个参数的TDDA系统动力学模型。
  • 模型涵盖两类微生物：SOB（XSOBX_{SOB}XSOB​）和厌氧氨氧化菌（XAMXX_{AMX}XAMX​），采用双项Monod型动力学。
  • 提出了分支硫代硫酸盐氧化途径：S2O32−S_2O_3^{2-}S2​O32−​首先被氧化为SO42−SO_4^{2-}SO42−​和S0S^0S0，随后S0S^0S0进一步被氧化为SO42−SO_4^{2-}SO42−​。

2. 模型校准与验证​

  • 从批次实验（A1和A2）中估计了9个关键参数，包括μSOB1\mu^1_{SOB}μSOB1​ = 5.47/d、μSOB2\mu^2_{SOB}μSOB2​ = 1.16/d、μSOB3\mu^3_{SOB}μSOB3​ = 0.0147/d、μSOB4\mu^4_{SOB}μSOB4​ = 0.0076/d、μAMX\mu_{AMX}μAMX​ = 0.0899/d、KS2O3K_{S_2O_3}KS2​O3​​ = 48.26 mgS/L、KS0K_{S^0}KS0​ = 1.77 mgS/L。
  • 在两种不同条件下（B1和B2）验证模型，分别获得R2>0.84R^2 > 0.84R2>0.84和R2>0.86R^2 > 0.86R2>0.86，证明模型能够描述氮和硫的动态转化。

3. 宏基因组学验证硫代硫酸盐氧化途径​

  • 宏基因组分析证实TDDA系统中硫代硫酸盐氧化遵循分支氧化途径（不完整Sox系统 + DsrAB + AprAB + Sat），而非完整Sox途径或S4IS_4IS4​I途径。
  • SoxCD基因缺失及DsrAB、AprAB、Sat基因的高丰度支持了实验中观察到的元素硫积累现象。

4. SOB与厌氧氨氧化菌的相互作用​

  • 在稳态条件下，不同S/N（0.5–3.5 gS/gN）和NH4+/NO3−NH_4^+/NO_3^-NH4+​/NO3−​（0.3–1.2 gN/gN）比值下，厌氧氨氧化菌占活性生物量的比例为12%至51%。
  • 在宽泛的操作条件范围内，厌氧氨氧化菌不会被SOB淘汰，这归因于SOB较低的生物量产率和生长速率。
  • 当NH4+/NO3−NH_4^+/NO_3^-NH4+​/NO3−​ > 0.6时，厌氧氨氧化是主导脱氮途径（贡献>73%的TN去除）。
  • 稳定协作的原因：(1) SOB中硝酸盐还原速率高于亚硝酸盐还原速率（μSOB1\mu^1_{SOB}μSOB1​是μSOB2\mu^2_{SOB}μSOB2​的4.8倍），确保向厌氧氨氧化持续供给亚硝酸盐；(2) 厌氧氨氧化菌对亚硝酸盐的亲和力（KNO2AMXK_{NO_2}^{AMX}KNO2​AMX​ = 0.05 mgN/L）远高于SOB（KNO2SOBK_{NO_2}^{SOB}KNO2​SOB​ = 0.43 mgN/L）。

5. 工艺操作的实践意义​

  • 推导出NH4+/NO3−NH_4^+/NO_3^-NH4+​/NO3−​（Y）与S/N（X）之间的线性关系：X=3.2−1.9×YX = 3.2 - 1.9 \times YX=3.2−1.9×Y，Y∈[0,0.9]Y \in [0, 0.9]Y∈[0,0.9]，定义了实现硫代硫酸盐完全氧化和高TN去除率（>98%）的最优操作区间。
  • 该模型为评估TDDA工艺可行性和调控硫代硫酸盐投加量以处理不同浓度硝酸盐和铵盐工业废水提供了实用工具。

Gemini 3.1 Pro:

  • 成功开发、校准并验证了TDDA工艺的动力学模型，该模型准确描述了氮和硫转化的动态过程。
  • 宏基因组学分析证实，TDDA系统中的硫代硫酸盐氧化遵循以单质硫为中间体的支链氧化途径。
  • 模型模拟表明，SOB和厌氧氨氧化细菌在各种底物浓度下都能稳定合作。
  • 厌氧氨氧化细菌对亚硝酸盐的竞争力强于SOB，只要铵盐充足，厌氧氨氧化仍然是主要的脱氮途径。
  • 所开发的模型可用于评估TDDA工艺的可行性，并优化处理含硝酸盐和铵盐的工业废水的硫代硫酸盐投加量。

Gemini 3.0 Pro:
