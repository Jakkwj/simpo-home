# Installtion

To calculate a **SIMPO** project, you need to download and install the calculation client, and then sign up and sign in to <a href={config.customFields.urlDashHome}>Dashboard</a>.

:::warning NOTE

<b>Please do not double-click to run the client after installation</b>, it will start automatically once calculation begins.

:::

{/* While running a calcualtion task, the client will stay in the system tray. Right click the client icon can get relevant informations. For instance, log file becomes very important if your calcualtion goes wrong, click **Log** and open the `run.log` to see what happened. */}

If the main program encounters an error during operation, the log file `SimpoClient_error.log` will automatically pop up. Please review the log contents to troubleshoot the issue.


:::tip Tip

You will not be able to run calculations if you don't install the client, but you can still sign in and use the other functions (i.e. editting you own repositories, browsing and cloning the public repositories).

:::


As a cross-platform web-based platform, **SIMPO** works on **Windows** and **Linux** operating
systems. Therefore, it can leverage the power of supercomputers, as their systems are usually
**Linux-based**.

          - This directory can be opened directly from the `Project`'s `Calculation` page under the `Simulation`/`Sensitivity`/`Uncertainty`/`Estimation` sections.
          - This directory stores the calculation data for each project, including raw calculation results, result figures, and runtime logs.
          - If the main program encounters an error, the log file `SimpoClient_error.log` will be generated in this directory to facilitate troubleshooting the cause of the error.
    </li>
