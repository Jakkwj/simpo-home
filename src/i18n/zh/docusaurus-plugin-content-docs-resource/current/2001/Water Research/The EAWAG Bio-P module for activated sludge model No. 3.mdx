<TabItem value="abstract" label="摘要" >

在经过校准的 ASM3 版本基础上，提出了一个用于预测强化生物除磷的附加模块。该模块使用了 ASM2d 中修改过的过程，但忽略了易降解基质的发酵。生物量衰减是以内源呼吸的形式建模的，与 ASM3 相同。糖原池和生物诱导的 P 沉淀未考虑在内。该模块通过各种批次实验、全规模污水处理厂和处理瑞士城市污水的中试厂的实验数据进行了系统校准。标准参数集允许模拟所有数据。

**Q：这篇论文试图解决什么问题？**

**Kimi:**
这篇论文介绍了一个额外的模块——EAWAG Bio-P模块，用于增强ASM3（Activated Sludge Model No. 3）模型的预测能力，特别是在增强生物除磷（EBPR）方面。ASM3是一个基本的活性污泥模型，主要用于预测市政废水处理系统中的氧气消耗、污泥产生、硝化和反硝化等过程，但它本身不包括生物除磷过程。EAWAG Bio-P模块的开发旨在填补这一空白，通过结合ASM3的校准版本，对生物除磷过程进行建模和预测。

具体来说，这篇论文试图解决的问题包括：

1. **生物除磷过程的建模**：通过引入EAWAG Bio-P模块，将生物除磷过程整合到ASM3模型中，以便更全面地模拟和预测废水处理过程中的氮、磷去除。

2. **参数校准和验证**：通过与不同批次实验、全尺寸污水处理厂和试验工厂处理瑞士市政废水的实验数据进行系统校准，确保模型参数的准确性和适用性。

3. **简化模型复杂性**：在保持模型准确性的同时，尽可能简化模型，例如忽略易于降解底物的发酵过程和额外的糖原池，以减少模型的复杂性和计算需求。

4. **适应不同操作条件**：通过模型校准，使其能够适应不同的操作条件和废水特性，如低COD负荷后的“周一效应”，以及不同污泥龄（SRT）下的处理效果。

5. **提高模型的预测能力**：通过校准和验证，确保模型能够准确预测不同动态效应，如短期内的磷释放和吸收、长期内的PAO（聚磷酸盐积累菌）的洗出和增长，以及低COD负荷后的出水磷酸盐峰值。

总的来说，这篇论文的目标是开发和验证一个能够准确模拟和预测生物除磷过程的模型，以便更好地理解和优化市政废水处理系统中的磷去除效率。

**Q：论文如何解决这个问题？**

**Kimi:**
论文通过以下步骤解决增强生物除磷（EBPR）的建模问题：

1. **模型开发**：开发了EAWAG Bio-P模块，这是一个附加模块，用于扩展ASM3模型以包括生物除磷过程。该模块基于ASM3的校准版本，并结合了ASM2d中的修改过程，但不包括易于降解底物的发酵过程。

2. **参数定义**：定义了四个额外的状态变量，除了ASM3中已经定义的13个组分，以模拟生物除磷过程。这些变量包括无机可溶性磷（SPO4）、聚磷酸盐积累菌（XPAO）、细胞内存储产物（XPHA）和聚磷酸盐（XPP）。

3. **过程定义**：详细描述了与生物除磷相关的11个额外过程，包括磷的吸收、存储和释放，以及聚磷酸盐积累菌（PAO）的有氧和缺氧生长条件。

4. **系统校准**：使用来自不同批次实验、全尺寸污水处理厂和试验工厂处理瑞士市政废水的实验数据，对EAWAG Bio-P模块进行了系统的校准。通过迭代方法，将高度相关的参数分组校准，并使用不同批次实验进行验证。

5. **动态效应模拟**：模型能够模拟不同的动态效应，例如正常的磷释放和吸收、PAO的洗出和增长，以及在低COD负荷时期（如周末、雨天、假期）后的出水磷酸盐峰值。

6. **模型验证**：通过与实验数据的比较，验证了模型的准确性。这包括对长期模拟和短期动态变化（如日变化）的验证。

7. **简化和适应性**：模型设计时考虑了简化复杂性，例如忽略了额外的糖原池，并针对瑞士市政废水的典型条件进行了适应性调整。

通过这些步骤，论文成功地将生物除磷过程整合到了ASM3模型中，并通过实验数据的校准和验证，提高了模型预测生物除磷效率的能力。这为废水处理工程师提供了一个强大的工具，以优化设计和操作，提高磷去除效率。
