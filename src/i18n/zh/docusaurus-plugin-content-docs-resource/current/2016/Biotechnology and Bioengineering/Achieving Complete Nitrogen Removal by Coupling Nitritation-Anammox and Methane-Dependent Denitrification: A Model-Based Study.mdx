<TabItem value="abstract" label="摘要" >

反硝化厌氧甲烷氧化（DAMO）过程的发现，使得利用厌氧消化池现场产生的甲烷从废水中完全去除氮成为可能。这项基于模型的研究调查了在膜生物膜反应器（MBfRs）中耦合亚硝化-厌氧氨氧化（Anammox）和甲烷依赖型反硝化以实现高效脱氮的机制和操作窗口。应用数学模型描述了Anammox细菌、DAMO古菌和DAMO细菌之间的微生物相互作用。该模型充分描述了含有Anammox-DAMO生物膜的MBfR在不同进水氮组成下的批次实验数据，证实了模型的有效性。因此，可以可靠地评估工艺参数对系统性能和微生物群落结构的影响。利用该模型综合研究了亚硝化产生的$$NO_2^-/NH_4^+$$比率、甲烷供应、生物膜厚度和总氮（TN）表面负荷的影响。结果表明，为了实现最大的TN去除率（超过99.0%），Anammox-DAMO生物膜系统的最佳亚硝化产水$$NO_2^-/NH_4^+$$比率约为1.0，且与TN表面负荷无关。相应的最佳甲烷供应量随着TN表面负荷的增加而增加，而相关的甲烷利用效率则降低。DAMO生物和Anammox细菌之间的合作在TN去除中起着关键作用。基于这些结果，还通过将模型与氨氧化细菌（AOB）和亚硝酸盐氧化细菌（NOB）过程整合，并控制模拟系统中的溶解氧（DO）浓度，测试了单级MBfR耦合亚硝化-Anammox-DAMO实现完全脱氮的概念验证可行性。发现在模拟条件下，当主体DO浓度约为0.17 $$g\ m^{-3}$$时，AOB、Anammox细菌和DAMO生物在生物膜中共存，可实现最大的TN去除。

**Q：这篇论文试图解决什么问题？**



------

**Claude Sonnet 4.6:**

本文试图解决污水处理中传统亚硝化-厌氧氨氧化（Anammox）工艺**氮素去除不完全**的问题。具体而言：

- 传统亚硝化-Anammox工艺无法去除污水中的硝酸盐，实际上会将约**20%的亚硝酸盐转化为硝酸盐**（如论文中方程1所示），导致无法实现完全总氮（TN）去除。
- 本文旨在确定将亚硝化-Anammox与反硝化厌氧甲烷氧化（DAMO）工艺耦合于膜生物膜反应器（MBfR）中实现完全脱氮的**最优操作条件**（如 $NO_2^-/NH_4^+$ 比值、甲烷供给量 $L_{CH_4}$、生物膜厚度 $L_f$ 及总氮表面负荷 $L_{TN}$）。
- 本文还解决了前期模型（Chen et al., 2014）在**亚硝酸盐存在及其对Anammox和DAMO微生物抑制**情况下缺乏验证的问题。
- 此外，本文还探索了将所有工艺（AOB亚硝化、Anammox和DAMO）集成于**单级MBfR**中的概念可行性，通过控制溶解氧（DO）浓度实现，以降低投资和运营成本。

**Q：论文如何解决这个问题？**



------

**Claude Sonnet 4.6:**

本文通过**基于数学模型的方法**解决上述问题，具体策略如下：

- **模型构建**：利用AQUASIM 2.1d软件建立多物种、多底物一维生物膜模型，整合了AOB、NOB、Anammox细菌、DAMO古菌和DAMO细菌的动力学。模型追踪6种颗粒态组分（$X_{AOB}$、$X_{NOB}$、$X_{Da}$、$X_{Db}$、$X_{An}$、$X_I$）和6种溶解态组分（$S_{NH4}$、$S_{NO2}$、$S_{NO3}$、$S_{N2}$、$S_{CH4}$、$S_{O2}$），并引入了Anammox和DAMO微生物的亚硝酸盐和氧气抑制动力学。

- **模型验证**：扩展模型（含亚硝酸盐抑制项）通过与实验室规模MBfR中Anammox-DAMO生物膜在不同进水氮组成下的**六组批次实验数据**（批次试验A–F）进行对比验证，模型预测值与实验值偏差不超过10%，无需进一步标定。

- **系统模拟情景**：设计了六个模拟情景（情景0–5）进行研究：
  - **情景0**：标准模拟，揭示生物膜分层结构和微生物群落组成。
  - **情景1**：研究亚硝化产生的 $NO_2^-/NH_4^+$ 比值（范围：0.25–4.0）对TN去除和微生物群落的影响。
  - **情景2**：研究甲烷表面负荷 $L_{CH_4}$（范围：0.016–0.250 g m$^{-2}$ d$^{-1}$）对系统性能的影响。
  - **情景3**：研究生物膜厚度 $L_f$（范围：200–1,400 μm）对TN去除的影响。
  - **情景4**：研究不同TN表面负荷下 $NO_2^-/NH_4^+$ 比值与 $L_{CH_4}$ 的联合效应（$L_{TN}$ = 0.41、0.68、0.95 g N m$^{-2}$ d$^{-1}$），共630次模拟运行。
  - **情景5**：通过控制液相DO浓度（范围：0.04–0.40 g m$^{-3}$），验证单级MBfR耦合亚硝化-Anammox-DAMO的概念可行性。

- **关键发现作为解决方案**：
  - 前置亚硝化的最优 $NO_2^-/NH_4^+$ 比值为**1.0**（与TN表面负荷无关），可实现最大TN去除率（>99.0%），比传统Anammox所需的1.32低，节省约10%的曝气能耗。
  - 最优甲烷供给量随TN表面负荷增加而增大，且仅需不到1%的现场产甲烷量即可实现完全脱氮。
  - 生物膜厚度至少需达到1,000 μm才能实现最优TN去除。
  - 对于单级MBfR，液相DO浓度约为**0.17 g m$^{-3}$**时，AOB、Anammox细菌和DAMO微生物可共存于生物膜中，实现最大TN去除。

**Q：论文做了哪些实验？**


------

**Claude Sonnet 4.6:**

实验工作主要用于提供**模型验证数据**（而非开发新工艺）。主要实验内容如下：

- **MBfR装置搭建与长期运行**：
  - 使用总体积为1,150 mL的实验室规模MBfR（含400 mL中空纤维膜、300 mL内部供气空间和450 mL外部液体空间）。
  - 中空纤维膜总表面积为1 m²。
  - MBfR以**序批式反应器（SBR）**模式运行，周期为1天，水力停留时间（HRT）为3天，温度22 ± 2°C，pH 7.0 ± 0.2。
  - 甲烷通过中空纤维膜内部以1.3 atm压力供给。
  - MBfR运行**超过400天**，进水氨氮（200–300 mg N L$^{-1}$）和硝酸盐氮（200–600 mg N L$^{-1}$）浓度逐步提高。
  - 400天以上运行后，**荧光原位杂交（FISH）**分析证实了DAMO古菌、DAMO细菌和Anammox细菌在生物膜中的共存与联合主导地位。

- **六组批次试验（A–F）用于模型评估**：
  - 长期SBR运行后，开展六组批次试验，评估模型在不同氮素进水组成下的适用性：
    - **批次试验A–D**：改变初始 $NO_2^-$ 和 $NH_4^+$ 浓度（无 $NO_3^-$）：
      - 试验A：$NO_2^-$ = 91.5 g N m$^{-3}$，$NH_4^+$ = 101.8 g N m$^{-3}$，持续8 h
      - 试验B：$NO_2^-$ = 110.4 g N m$^{-3}$，$NH_4^+$ = 173.4 g N m$^{-3}$，持续4.5 h
      - 试验C：$NO_2^-$ = 78.3 g N m$^{-3}$，$NH_4^+$ = 28.9 g N m$^{-3}$，持续6 h
      - 试验D：$NO_2^-$ = 88.1 g N m$^{-3}$，$NH_4^+$ = 36.4 g N m$^{-3}$，持续10 h
    - **批次试验E–F**：改变初始 $NO_3^-$ 和 $NH_4^+$ 浓度（无 $NO_2^-$）：
      - 试验E：$NO_3^-$ = 202.7 g N m$^{-3}$，$NH_4^+$ = 59.2 g N m$^{-3}$，持续30 h
      - 试验F：$NO_3^-$ = 161.8 g N m$^{-3}$，$NH_4^+$ = 86.7 g N m$^{-3}$，持续25 h
  - 每次批次试验期间，定期取液体样品，使用**Lachat QuikChem8000流动注射分析仪**测定 $NH_4^+$、$NO_2^-$ 和 $NO_3^-$ 浓度。
  - 所有批次试验中均以1.3 atm压力持续供给甲烷。
  - 将实验测得的氮素消耗曲线与模型预测结果进行对比，以验证模型的适用性。

<TabItem value="exploration" label="探索" >

**Q：有什么可以进一步探索的点？**


------

**Claude Sonnet 4.6:**


- **单级MBfR的实验验证**：单级MBfR耦合亚硝化-Anammox-DAMO的概念可行性仅通过模型模拟进行了验证。作者明确指出"此类耦合亚硝化-Anammox-DAMO生物膜系统的实际实施仍需进一步实验验证"。未来工作应实际搭建并运行此类单级系统。

- **含DO情况下的模型改进**：由于目前缺乏液相含DO条件下新耦合工艺的实验数据，模型尚未在此条件下进行测试。一旦单级系统实验开发完成，应利用新生成的数据对模型进行进一步标定和验证。

- **好氧甲烷氧化**：论文因其在模拟条件下影响可忽略而未将好氧甲烷氧化纳入模型。然而，在DO浓度较高的系统中，好氧甲烷氧化可能变得显著，应将其纳入模型。

- **异养细菌**：模型未考虑异养细菌，因为系统无外加有机碳。在实际污水处理场景中，有机碳可能存在，应将异养过程纳入模型以更好地反映实际情况。

- **甲烷排放与温室气体控制**：论文指出过量甲烷供给会导致甲烷浪费和潜在温室气体排放。在保持高TN去除效率的同时最小化甲烷排放的策略值得进一步研究，包括尾气捕集与利用。

- **能量权衡分析**：论文简要提及最优 $NO_2^-/NH_4^+$ 比值1.0可节省约10%曝气能耗，但需要额外甲烷（约72 g COD m$^{-3}$）。有必要对所提工艺与传统亚硝化-Anammox进行更全面的技术经济分析和生命周期评估（LCA）。

- **生物膜厚度控制策略**：论文确定了生物膜厚度是关键参数，但仅简要提及剪切力和气体鼓泡作为控制方法。应进一步深入研究生物膜厚度控制策略（如机械剪切、反冲洗等）及其对微生物群落结构的影响。

- **关键生物动力学参数的敏感性**：敏感性分析确定 $\mu_{An}$、$\mu_{Db}$ 和 $Y_{An}$ 是最敏感的参数。未来工作应重点针对不同工程系统准确测定或标定这些参数，以提高模型预测能力。

- **放大与实际污水应用**：所有实验和模拟均在实验室规模下以合成污水进行。耦合亚硝化-Anammox-DAMO系统在实际污水（含复杂有机物、变化氮负荷和抑制性物质）条件下的中试或全规模性能需要进一步研究。

- **氧化亚氮（$N_2O$）和一氧化氮（NO）排放**：论文未涉及耦合工艺中 $N_2O$ 和NO温室气体的产生问题。鉴于这些气体的环境重要性，应研究Anammox-DAMO生物膜系统中其产生与减排问题。

**Q： 总结一下论文的主要内容。**

**Claude Sonnet 4.6:**

本文提出了一项**基于数学模型的研究**，探讨在膜生物膜反应器（MBfR）中耦合亚硝化、Anammox和反硝化厌氧甲烷氧化（DAMO）工艺实现污水完全脱氮的机制与操作窗口。主要内容如下：

- **工艺概念**：传统亚硝化-Anammox工艺将约20%的亚硝酸盐转化为硝酸盐，无法实现完全TN去除。通过在MBfR中将DAMO古菌（利用 $CH_4$ 将 $NO_3^-$ 还原为 $NO_2^-$）和DAMO细菌（利用 $CH_4$ 将 $NO_2^-$ 还原为 $N_2$）与Anammox细菌耦合，可利用厌氧消化产生的现场甲烷实现完全脱氮。

- **数学模型**：在AQUASIM 2.1d中建立多物种、多底物一维生物膜模型，整合了AOB、NOB、Anammox细菌、DAMO古菌和DAMO细菌的动力学，包含亚硝酸盐和氧气抑制项。模型经六组批次实验数据验证，偏差不超过10%。

- **生物膜分层**：在Anammox-DAMO MBfR中，DAMO微生物主导靠近膜基底的区域（0–400 μm，甲烷可及），Anammox细菌主导靠近液相的外层（400–1,000 μm，氨氮和亚硝酸盐可及）。这种逆扩散分层是系统高效运行的关键。

- **最优 $NO_2^-/NH_4^+$ 比值**：前置亚硝化产生的最优 $NO_2^-/NH_4^+$ 比值为**1.0**（与TN表面负荷无关），可实现>99.0%的TN去除率。这低于传统Anammox所需的1.32，节省约10%曝气能耗。理论依据是DAMO古菌将Anammox产生的20%硝酸盐循环还原为亚硝酸盐。

- **甲烷供给优化**：最优甲烷表面负荷随TN表面负荷增加而增大，甲烷利用效率随之降低（从 $L_{TN}$ = 0.41 g N m$^{-2}$ d$^{-1}$ 时的93.0%降至 $L_{TN}$ = 0.95 g N m$^{-2}$ d$^{-1}$ 时的73.8%）。实现完全脱氮仅需不到1%的现场产甲烷量。

- **生物膜厚度**：最优TN去除率（>99.0%）需要生物膜厚度至少约1,000 μm。较薄的生物膜由Anammox细菌主导，DAMO活性不足。

- **单级MBfR可行性**：通过将液相DO控制在约**0.17 g m$^{-3}$**，单级MBfR耦合亚硝化-Anammox-DAMO可实现最大TN去除率90.7%，AOB、Anammox细菌和DAMO微生物在生物膜中共存，NOB在此DO水平下受到抑制。

- **微生物竞争**：在两级系统中，Anammox细菌与DAMO古菌的协作是TN去除的关键；在单级系统中，DAMO细菌在TN去除中发挥重要作用。过量甲烷供给会使DAMO细菌占优并淘汰Anammox细菌，降低TN去除效率。
