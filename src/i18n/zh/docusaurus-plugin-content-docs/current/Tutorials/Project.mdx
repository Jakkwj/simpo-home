# Project

Project 是将 BioModel 和 DataSet 结合在一起并进行求解（模拟和评估）的地方。

## <a id="Create"></a>新建

创建 Project 有几种方法（从下面的 3 种方法中选择一种）：

### <a id="way_1"></a>1. 空白新建

创建一个空的 Project，编辑并解析保存后使其生效。

  - 如果您没有指定 BioModel/DataSet，将相应地创建一个空的 BioModel/DataSet。
  - 如果 BioModel 的 components 与 DataSet 的 targets 完全相同，则 Project 的 detial 将自动生成。
  - 否则，您必须指定 BioModel 的 components 与 DataSet 的 targets 之间的转换关系。

## <a id="Solve"></a>求解

模拟和评估只需几次鼠标点击即可启动：
  - 选择计算引擎：Simulaiton, Sensitivity, Uncertainty 或 Estimation。
  - 点击 START 按钮并等待结果，SIMPO 将在计算过程完成后绘制所有内容。

## <a id="Edit"></a>编辑
在 Project 中，您可以直接编辑对应的 BioModel 和 DataSet，然后在 Solution 标签页中编辑求解参数。在 Calculation 标签页中点击 START 按钮开始计算后，即可得到计算结果。


## <a id="Release"></a>发布

在 Version 选项卡中可以发布不同的版本，每个版本都有一个版本号。
