# DataSet


DataSet 是存放您测量数据的地方。

:::tip 流程图

添加反应及其链接关系（入流、流量和连接）后，流程图将自动生成。 您可以在此流程图中点击并编辑污染物目标、反应器、入流、流量和连接。

:::


## <a id="Create"></a>新建

创建 DataSet 有几种方法（从下面的 6 种方法中选择一种）：

### <a id="way_1"></a>1. 空白新建

创建一个空的 DataSet，编辑并解析保存后使其生效。
- 添加新 targets。
- 添加新 tanks。
- 可选：添加新的链接（入流、流量和连接）。如果您只使用独立的反应器，则此步骤不是必需的。
