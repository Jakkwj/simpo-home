# BioModel



BioModel 是您建立自己的生化模型的地方。

## <a id="Create"></a>新建

创建 BioModel 有几种方法（从下面的 5 种方法中选择一种）：

### <a id="way_1"></a>1. 空白新建

创建一个空的 BioModel，编辑并解析保存后使其生效。
- 添加新 components。
- 添加新 parameters。
- 添加新 processes（在 Matrix选项卡中）。

## <a id="Edit"></a>编辑

如果 parameter 是速率参数（如 `u_max` 的单位为`/day`），其单位**必须以**：``/d``, ``d-1``, ``/day``, ``day-1``, ``1/d``, ``1/day``, ``/h``, ``h-1``, ``/hour``, ``hour-1``, ``1/h``, ``1/hour``, ``/m``, ``m-1``, ``/minute``, ``minute-1``, ``1/m``, ``1/minute``, ``/s``, ``s-1``, ``/second``, ``second-1``, ``1/s`` or ``1/second`` 为结尾，这将有助于在整个计算过程中统一时间单位。
