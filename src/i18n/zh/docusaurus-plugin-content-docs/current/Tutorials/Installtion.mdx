# 安装

作为一个跨平台的网络平台，**SIMPO** 可在 Windows、Linux 和 Mac（待定）操作系统上运行
。因此，它可以利用超级计算机的强大性能，因为它们的系统通常是基于 Linux 的。

要计算 **SIMPO** 项目，您需要下载并安装计算客户端，然后注册并登录到 <a href={config.customFields.urlDashHome}>Dashborad</a>。

:::warning 注意

计算开始后，客户端将自动运行，请<b>不要双击运行客户端</b>。

:::


在运行计算任务时，客户端会停留在系统托盘中。右键点击客户端图标可以获取相关信息。例如，如果计算出错了，日志文件将就变得非常重要，点击 **Log**，打开 `run.log` 查看错误。

:::tip 提示

如果不安装客户端，计算将无法运行，但您仍可登录并使用其他功能（如编辑自己的私有库、浏览和克隆公开库）。

:::
