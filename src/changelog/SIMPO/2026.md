## 2026-02-01

- **Fixed**
  - When the plotting function encounters a `BrokenProcessPool` error, it will use the main process to continue plotting.
  - Fixed a rare saving error in BioModel (caused by invisible spaces in the frontend).

## 2026-01-02

- **Added**
  - Added support for **pH** calculation; related parameters are configured in `BioModel.Ionization`.
  - The local Real-Time Plotting window now supports switching images using the left/right mouse buttons or keyboard keys, and the aspect ratio is maintained during switching.

- **Changed**
  - **Improved Euler** will no longer be supported. Equivalent computational accuracy can be achieved (more quickly) by reducing the **Step Tolerance** of **Forward Euler**.

- **Improved**
  - Optimized the Real-Time Plotting function for reduced memory and CPU consumption.
  - Removed scientific notation from axes in locally saved **SVG** images.

- **Fixed**
  - `Inflow` Figure is now displayed correctly.
  - Fixed an issue where the last point in locally saved **SVG** images was sometimes not drawn correctly.
  - The Real-Time Plotting window now refreshes correctly.
  - The taskbar icon for the Real-Time Plotting window now displays correctly.
  - Fixed some known bugs.
