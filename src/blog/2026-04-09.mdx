-   `TimeUnit` is fixed to "day".
-   `Measured` is calculated up to a maximum of 10,000 days.
-   Only the first and last rows of `Measured` are kept.
-   Only the first row of `Inflow` and `Flow` is kept.
-   Each round's calculation result is truncated to 4 decimal places (saved to an auxiliary variable, without affecting the original calculation result). If this truncated result remains the same for 10,000 consecutive times, the calculation terminates, yielding a steady-state solution.
